chapter1
什么是spring
spring是java企业级开源解决框架，spring框架的核心功能能被用来开发任意java程序，也有一些扩展是java web平台顶级开源方案。框架的目标是让java更易于使用。
spring框架的长处是什么
轻量
spring是轻量的，基础版本的spring框架只有2MB
控制反转 IOC
由对象提供提供其依赖关系，而不是创建或查找依赖关系
高级模块不应该依赖于低级模块，两者都应该取决于抽象
抽象不应该依赖于细节，细节应取决于抽象
切面 AOP
面向切面变成，将辅助支持功能于主程序业务逻辑隔离出来的变成范式。AOP于oop相互补充，通过oo实现项目，通过方面分别处理代码中的横切关注点
容器 container
spring 管理程序对象容器的生命周期
MVC Framework mvc框架
spring web框架有着非常良好的设计
Transaction Management 事务管理
spring提供了一组书屋管理接口能降级到 本地食物或者扩容到全局食物
Exception Handling 异常拦截
spring提供了一组方便的 API去转换比较特俗的异常，比如被 JDBc ,Hibernateor jdo扔出来的非受检异常
